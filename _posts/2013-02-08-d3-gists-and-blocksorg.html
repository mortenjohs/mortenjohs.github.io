--- layout: post title: D3, gists, and bl.ocks.org date: '2013-02-08T22:57:00.001+01:00' author: Morten tags: - Programming - gist - source code - web - github - visualization modified_time: '2013-02-08T22:57:58.386+01:00' blogger_id: tag:blogger.com,1999:blog-6967032375013519080.post-8198465017175052950 blogger_orig_url: https://m635j520.blogspot.com/2013/02/d3-gists-and-blocksorg.html --- Yesterday I stumbled upon a very useful tool while (trying to) teach(ing) myself D3, namely http://bl.ocks.org. This lets you access, and preview, your webby gists in a browser - just by browsing to http://bl.ocks.org/<gist-id>. Most handy!
(The only minor down side is that there seems to be a slight latency if you update your gist before it updates on bl.ocks.org.)
